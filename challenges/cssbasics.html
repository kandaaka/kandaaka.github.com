Inheritance in CSS

Inheritance is when one element inherits the properties of its parent element if there is no specified property for the element. Examples of inherited properties are color, margin, font size, and padding. Border is an example of a property that isn't inherited from the parent element.

Specifity in CSS

Specificy is when an it is specified which element will receive the property. This can be determined by using class or id. It is mainly used when there are two or more CSS rules for a specific element. The browser follows the rules and determines which one is more specific.There are two types of selectors the multiple selector and the compound selector. The multiple selector is when several elements have the same property.This type of selection allows you to minimize the code lines and the elements are separated by a comma (ie. h1, h2, p). A compound selector gets higher priority when deciding what property the element gets (ie. nav ul li a). ID and Class Attributes

ID can only be used once in the html, while the class attribute will allow you to use it as many times and to any element. All the elements with class will get the same properties.
