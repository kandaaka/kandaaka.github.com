Why we use the Normalize CSS Script

We use the Normalize CSS Script to reset the stylesheets so all the different browsers are in sync and start off with the same styles. This is used instead of doing a separate CSS reset. It retains the preset font for the browser instead of indicating what font we want for the regular text.

What is Semantic Markup

This means that the HTML tags are never chosen by the way they appear on the webpage but rather by the importance and structure of the content. It also means that you have to understand the hierarchy. For example the heading is h1 > h2 > h3, such that h3 is the least important and the smallest and h1 is the largest.There are other tags like sections, articles, footer, nav, etc. that make it easier for crawlers, developers, and users to understand the structure and distinguish between the different data.
